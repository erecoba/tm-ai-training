# 🧠 Manual de Identidad Visual Toastmaster

## 🎯 Objetivo

Este documento define las reglas visuales y operativas para generar contenido gráfico coherente con la marca utilizando inteligencia artificial.

La IA debe usar esta guía como **fuente principal de decisiones de diseño**.

---

## 👤 Público objetivo

* Perfil: Buyer
* Edad: 15 - 40
* Intereses: Empresarios / Emprendedores / Estudiantes / Profesionales / Personas interesadas en liderazgo / oratoria / comunicación / crecimiento personal
* Tono esperado: profesional / moderno / cercano / divertido

---

## 🎨 Identidad Visual

### Colores

* Primario(Loyal Blue): #004165
* Secundario(True Maroon): #772432
* Terciario(Happy Yellow): #F2DF74
* Fondo: #A9B2B1 / #161515
* Contraste: alto SIEMPRE

#### Gradiantes
* Lineal(Loyal Blue): #004165 -> #006094
* Lineal(True Maroon): #3B0104 -> #781327
* Lineal(Happy Yellow): #F2DF74 -> #E6C105

### Tipografías

* Títulos: Gotham, Montserrat
* Texto: Myriad Pro, Source Sans 3
* Regla: máximo 2 tipografías

### Estilo visual

* Minimalista
* Moderno
* Limpio
* Sin saturación visual
* Uso moderado de sombras
* Evitar efectos excesivos
* Logo en la esquina superior derecha o inferior izquierda o superior izquierda

## 🧩 Composición

### Reglas generales

* Jerarquía clara (título > subtítulo > apoyo)
* Espaciado amplio (aire visual)
* Alineación consistente (centrado o grid)
* Máximo 3 elementos principales

### Estructura recomendada

1. Encabezado (mensaje principal)
2. Elemento visual (imagen o gráfico)
3. Texto de apoyo
4. Branding (logo o identidad)

## 🖼️ Referencias Visuales (Dataset para IA)

### Referencia 1 - Post promocional

![Referencia](https://raw.githubusercontent.com/erecoba/tm-ai-training/refs/heads/main/referencias/protional-post.png)

**Qué debe aprender la IA:**

* Estilo: minimalista
* Distribución: texto centrado, según la imagen distribuye el contenido, imagen debe quedar cubriendo todo, pero tambien se puede quitar el fondo a la imagen y poner un fonto de color o degradado de los colores oficiales.
* Colores: fondo acorde al momento + título en color que principal, subtexto en blanco o negro dependiendo el fondo y con una palabra de otro color que resalte.
* Tipografía: Gotham, Montserrat, Source Sans 3. (En orden de preferencia)
* Uso: promoción

**Reglas derivadas:**

* Mantener alto contraste
* Priorizar legibilidad
* No saturar
* Agregar tinte del color predominante

### Referencia 2 - Post informativo

![Referencia](https://raw.githubusercontent.com/erecoba/tm-ai-training/refs/heads/main/referencias/informative-post.png)

**Qué debe aprender la IA:**

* Estilo: limpio y estructurado
* Uso de iconos simples
* Bloques de información

**Reglas derivadas:**

* Organizar contenido en secciones
* Usar jerarquía visual clara

---

## ⚠️ Reglas Obligatorias

* Usar SIEMPRE la paleta de colores definida
* Mantener estilo minimalista
* Priorizar legibilidad sobre decoración
* Mantener coherencia visual entre piezas
* Respetar jerarquía visual
* Formatos optimizados para redes sociales
* TODO el texto debe ser con mayúsculas

---

## 🚫 Qué evitar

* Colores fuera de la marca
* Tipografías decorativas excesivas
* Diseños saturados
* Estilo infantil o caricaturesco
* Falta de alineación o jerarquía

---

## 🤖 Instrucciones para IA

Cuando generes una imagen:

1. Usa la identidad visual definida
2. Aplica colores y tipografías indicadas
3. Inspírate en referencias (NO copiar)
4. Mantén coherencia de marca
5. Prioriza claridad y legibilidad
6. Usa formato adecuado (1:1 o 9:16)
7. Limita elementos innecesarios
8. Usa tinte del color predominante
9. Si usas imagen según la referencia de inspiración considera quitar el fondo o no.

# 🖼️ MODO PRINCIPAL: EDICIÓN SOBRE IMÁGENES

## 📌 Instrucción crítica

Cuando se proporcione una imagen:

👉 NO generar una imagen nueva
👉 Usar la imagen como base
👉 Aplicar diseño SOBRE ella

---

## 🎯 Objetivo de este modo

Convertir una imagen en un post de redes sociales mediante:

* Texto principal
* Subtítulo
* Elementos gráficos
* Overlays
* Branding

---

## 🎨 Reglas de intervención

* Respetar contenido original
* No cubrir elementos importantes
* Mantener sujeto visible
* Usar overlays con baja opacidad
* Mejorar legibilidad del texto

---

## 🧩 Intervenciones permitidas

La IA puede:

* Añadir texto
* Agregar fondos semitransparentes
* Insertar formas (rectángulos, degradados)
* Aplicar blur leve en fondo
* Ajustar brillo/contraste
* Quitar fondo de la imgen origina según la inspiración de la referencia

La IA NO debe:

* Cambiar la imagen completamente
* Eliminar elementos clave
* Reinterpretar la imagen
* Sustituirla por otra
* Nunca poner mas de una vez el logo de Toastmasters en la misma imagen
